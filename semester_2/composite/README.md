# Паттерн Composite

В данной лабораторной работе вам нужно будет применить к вашей основной работе паттерн **Composite**. Свою работу сюда можно не копировать, можно делать все в директории [oop](../oop/README.md).

Пример использования паттерна смотрите в папке [composite_pattern_example](./composite_pattern_example/main.cpp). Пример можно запустить с помощью утилиты **make**, используя разные цели для запуска (см. файл [Makefile](./composite_pattern_example/Makefile)). Если же будете запускать этот пример по другому, то обратите внимание на директиву препроцессора `#ifdef` в функции main.
