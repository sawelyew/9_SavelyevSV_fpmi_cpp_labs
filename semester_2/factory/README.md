# Паттерн Factory

В данной лабораторной работе вам нужно будет использовать в вашей основной работе паттерн **Factory**. Этот может быть реализовано двумя способами (в работе нужно будет продемонстрировать оба!):
- с помощью классической реализации паттерна (через наследование);
- с помощью "обобщённого конструктора" (static метод).

Свою работу сюда можно не копировать, можно делать все в директории [oop](../oop/README.md).
