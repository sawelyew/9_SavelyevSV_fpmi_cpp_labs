# Домашние животные

Каждая строка текстового файла *classes.txt* содержит информацию о **домашнем животном**. Там содержится: 
- имя владельца (однозначно идентифицирует владельца);
- вид животного (кот, собака и т.д. – хотя бы 4 вида);
- кличка (может отсутствовать);
- возраст в годах.

Допускается ситуация, когда несколько строк файла совпадают (например, безымянные муравьи в муравьиной ферме). Все поля разделены точкой с зарятой `;`, отсутствие клички обозначается точками с запятой, стоящими подряд (`;;`). 

> У каждой объекта класса должен быть свой **уникальный числовой идентификатор**, не изменяющийся после создания объекта!

Животное можно покормить (`feed`), также с ним можно поиграть (`play`), в результате чего каждый вид животного должен по особому на это отреагировать!

Вам необходимо:
1) Прочитать содержимое файла в один контейнер(подумайте, что это может быть за контейнер!);
2) после этого предусмотреть выполнение следующих запросов:
    - С помощью **одного цикла** покормить всех животных конкретного владельца (вводится пользователем).
    - С помощью **одного цикла** поиграть со всеми животными конкретного владельца (вводится пользователем).
    - Посчитать количество животных каждого вида.
    - Посчитать количество различных видов животных у каждого владельца.
    - Для конкретного вида животного (вводится пользователем) вывести наиболее популярную кличку.
    - Вывести информацию о возрасте самого старого и самого молодого животного каждого вида.

Результаты вывести на экран в форме удобной для чтения (для запросов использовать меню). Предусмотреть ситуации, когда пользователь вводит некорректные данные.
