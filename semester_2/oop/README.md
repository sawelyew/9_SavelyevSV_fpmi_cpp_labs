# ООП. Наследование. Полиморфизм. Абстрактные классы

В данной лабораторной работе нужно будет создать иерархию из абстрактного базового класса и нескольких неабстрактных классов наследников. Варианты распределяются следующим образом ($N$ – порядковый номер в группе):
- $N \mod 3 = 0 \Rightarrow$ [Домашние животные](./animals.md)
- $N \mod 3 = 1 \Rightarrow$ [Аудиторные занятия](./classes.md)
- $N \mod 3 = 2 \Rightarrow$ [Функции](./functions.md)

Поля и методы базового и производных классов продумать самостоятельно (будет оцениваться!), какие-то поля и методы уже будут описаны в условии. Также использовать обработку исключений.

Работу можно выполнять прямо в этой директории.
