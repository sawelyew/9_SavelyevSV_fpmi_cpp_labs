# Структуры. Паттерн Iterator

Данная лабораторная работа делится на несколько частей:

- Необходимо реализовать в зависимости от варианта шаблонный класс контейнерного типа (см. файл [collectionsmd](./collections.md)). Структуры реализуются на базе динамического массива. Данные могут быть произвольного типа (в том числе и пользовательского). **Номер варианта**: $N \mod 5 + 1$, где $N$ – порядковый номер в группе.
- К своей структуре данных применить паттерн **Iterator**, реализовав его через методы `First`, `Next`, `CurrentItem` и `IsDone` (Пример использования смотрите в папке [iterator_pattern_example](./iterator_pattern_example/main.cpp)).
- Разработать в зависимости от варианта внешнюю функцию (см. файл [outer_functions.md](./outer_functions.md)) для демонстрации работы паттерна (подумайте, что эта функция будет принимать в качестве аргументов!). **Номер варианта**: $(M - N) \mod 6 + 1$, где $M$ – количество человек в группе, а $N$ – порядковый номер в группе.

P.S. Обратите внимание, что **Iterator** – это **паттерн поведения**, т.е. это **НЕ ВНУТРЕННЯЯ СТРУКТУРА** класса-контейнера!
