# Список коллекций

Для каждой коллекции будут написаны требования в виде набора методов и операторов, которые обязательны к реализации. Но не запрещается добавлять свои методы для работы, если они будут нужны.

### 1. `Deque` – двусторонняя очередь.
Методы и операторы:
- конструкторы: по умолчанию, копирования, перемещения, `initializer_list<T>`
- деструктор
- `operator=`: копирующий и перемщающий
- `operator[]`
- `operator<<`
- `Size` – количество элементов
- `Clear` – удалить все элементы
- `Empty`– является ли дек пустым
- `Front`– получить **значение** первого элемента
- `Back` – получить **значение** последнего элемента
- `PushFront` – добавить новый элемент в начало дека
- `PopFront` – удалить элемент из начала дека
- `PushBack` – добавить новый элемент в конец дека
- `PopBack` – удалить элемент из конца дека
